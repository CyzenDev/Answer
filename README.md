# 江铜Home答题辅助

## 项目依赖

本软件依赖Auto.js 4.1.1 Alpha2和自制脚本

## 功能列表

1. 4种答案显示模式
    ```
   1. 仅选项
   2. 选项+内容
   3. 题目+选项
   4. 题目+选项+内容
    ```
2. 自动解析剪贴板
3. 自动生成AI提示词
4. 支持保存新题目到题库

## 题型支持

1. 闯关答题
2. 有奖答题

## 使用方法

1. 导入题库（没有可导入[题库模板](题库模板.xlsx)），支持xls和xlsx文件
2. 开启小窗模式
3. 执行脚本
4. 点击窗口，自动获取答案，如遇到没有答案的题自动生成AI提示词
